# **App Name**: TogetherApp

## Core Features:

- User Sync: Enable users to connect with a unique CoupleID through invite/accept.
- Weather at Partner's Location: Display current weather conditions at the partner's location using an API.
- Time at Partner's Location: Display the current time and timezone at the partner's location.
- Distance Calculation: Calculate and display the distance between the two users based on their locations.
- Countdown to Next Date: Allow users to set a future date and display a countdown to that date.
- Heartbeat Notes: Enable each user to leave a single editable note for their partner.
- AI-Powered Date Suggestion: Using a tool that suggests date ideas, taking into account their distance and mutual interests.

## Style Guidelines:

- Primary color: Muted violet (#946AA0) for calmness and connection.
- Background color: Light gray (#F0F0F5), a very pale tint of the primary color, for subtlety.
- Accent color: Soft rose (#D6A2A9) for gentle highlights and important actions.
- Body and headline font: 'PT Sans', a humanist sans-serif for readability and warmth.
- Use soft, rounded icons for a gentle, approachable feel.
- Implement a cascading tile layout for the 'Always in Sync' section to present information in an organized and visually appealing manner.
- Use smooth, subtle transitions and animations to enhance the feeling of connectedness.
